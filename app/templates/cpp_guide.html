Guía de C++

Introducción a C++

Al igual que con Logo, donde le decimos a una tortuga qué hacer, en C++ le decimos a la computadora cómo hacer las cosas, ¡pero de una forma un poco diferente!

Operaciones Básicas

En C++ podemos hacer operaciones matemáticas. Aquí tienes algunas instrucciones básicas:

  • int suma = a + b; - Suma los valores de "a" y "b".
  • int resta = a - b; - Resta "b" de "a".

Ejemplo:


int a = 5;

int b = 3;

int resultado = a + b; // El resultado es 8

Tomando Decisiones

Al igual que en Logo, donde la tortuga decide si moverse o girar, en C++ podemos decidir qué hacer con "if".

  • if (condición) { instrucciones } - Si la condición es verdadera, realiza las instrucciones dentro de las llaves.

Ejemplo:


int numero = 10;

if (numero > 5) {

    // Esta parte se ejecutará porque 10 es mayor que 5.

}

Repetir Instrucciones

¿Y si queremos hacer algo muchas veces? ¡Usamos bucles!

  • for (inicio; condición; actualización) { instrucciones } - Repite las instrucciones mientras la condición sea verdadera.

Ejemplo:


for (int i = 0; i < 5; i++) {

    // Esta parte se ejecutará 5 veces.

}

Creando Funciones

Las funciones en C++ son como mini-programas que podemos usar una y otra vez. 


void saludo() {

    cout << "¡Hola, Mundo!";

}


// Ahora podemos usar nuestra función "saludo" para mostrar el mensaje.

saludo();  // Esto imprimirá "¡Hola, Mundo!" en la pantalla.

¡Y eso es solo el comienzo de lo que podemos hacer con C++!

Tomando Decisiones con Condicionales

En C++, podemos tomar decisiones en nuestro código usando condicionales. Imagina que le dices a la computadora: "Si algo es cierto, entonces haz esto; de lo contrario, haz eso otro".

  • if (condición) { instrucciones } - Si la condición es verdadera, realiza las instrucciones dentro de las llaves.
  • else { instrucciones } - Si la condición del "if" anterior no es verdadera, realiza estas instrucciones.

Ejemplo:


int edad = 12;
if (edad >= 18) {
    cout << "Eres un adulto.";
} else {
    cout << "Aún eres joven.";
}
// Esto imprimirá "Aún eres joven." en la pantalla.

Repetir Instrucciones con Bucles

Los bucles nos permiten repetir instrucciones varias veces. Así es como le decimos a la computadora que haga algo muchas veces sin tener que escribirlo una y otra vez.

  • for (inicio; condición; actualización) { instrucciones } - Repite las instrucciones mientras la condición sea verdadera.
  • while (condición) { instrucciones } - Continúa repitiendo las instrucciones mientras la condición sea verdadera.

Ejemplos:


// Bucle for
for (int i = 1; i <= 5; i++) {
    cout << "Número: " << i << endl;
}
// Esto imprimirá los números del 1 al 5 en la pantalla.

// Bucle while
int contador = 1;
while (contador <= 3) {
    cout << "Repetición: " << contador << endl;
    contador++;
}
// Esto imprimirá "Repetición: 1", "Repetición: 2" y "Repetición: 3".
