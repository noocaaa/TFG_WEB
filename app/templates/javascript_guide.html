Guía de JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que le da vida a nuestras páginas web. Con él, podemos crear interactividad, animaciones, juegos y mucho más. ¡Es como la magia detrás de muchos sitios web!

Variables y Datos

Las variables son como cajas donde guardamos información. Esta información puede ser un número, un texto, una lista y otros tipos de datos.

Ejemplo:


var nombre = "Ana";
var edad = 12;

El código anterior guarda el texto "Ana" en la variable "nombre" y el número 12 en la variable "edad".

Funciones

Las funciones son bloques de código que podemos "llamar" o "ejecutar" cuando queramos. Nos ayudan a organizar y reutilizar código.

Ejemplo:


function saludo() {
    alert("¡Hola, bienvenido!");
}

saludo();  // Esto mostrará un mensaje de bienvenida.

La función "saludo" muestra un mensaje de bienvenida cuando la llamamos.

Eventos

Con JavaScript, podemos hacer que algo suceda cuando ocurre un evento, como un clic en un botón.

Ejemplo:


var boton = document.getElementById("miBoton");

boton.addEventListener("click", function() {
    alert("¡Has hecho clic en el botón!");
});

El código anterior muestra un mensaje cuando haces clic en un botón con el ID "miBoton".

Condicionales

Los condicionales nos permiten tomar decisiones en nuestro código. Si se cumple una condición, hacemos algo; si no, hacemos otra cosa.

Ejemplo:


var numero = 10;

if (numero > 5) {
    alert("El número es mayor que 5.");
} else {
    alert("El número es 5 o menor.");
}

El código anterior muestra un mensaje dependiendo del valor de la variable "numero".
