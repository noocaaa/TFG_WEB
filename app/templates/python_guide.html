Guía de Python

Introducción a Python

Python es un lenguaje de programación de alto nivel conocido por su simplicidad y legibilidad. Se utiliza en una variedad de aplicaciones, desde desarrollo web hasta inteligencia artificial.

Operaciones Básicas

Al igual que en C++, Python también nos permite hacer operaciones matemáticas básicas.

  • suma = a + b - Suma los valores de "a" y "b".
  • resta = a - b - Resta "b" de "a".

Ejemplo:


a = 5
b = 3
resultado = a + b  # El resultado es 8

Tomando Decisiones con Condicionales

Python, al igual que C++, tiene estructuras condicionales para tomar decisiones en el código.

  • if condición: - Si la condición es verdadera, realiza las instrucciones siguientes.
  • elif otra_condición: - Si la condición anterior no se cumple, pero esta sí, realiza estas instrucciones.
  • else: - Si ninguna de las condiciones anteriores se cumple, realiza estas instrucciones.

Ejemplo:


edad = 12
if edad >= 18:
    print("Eres un adulto.")
else:
    print("Aún eres joven.")
# Esto imprimirá "Aún eres joven."

Repetir Instrucciones con Bucles

Python tiene estructuras de bucle que nos permiten repetir instrucciones.

  • for elemento in lista: - Repite las instrucciones para cada "elemento" en "lista".
  • while condición: - Continúa repitiendo las instrucciones mientras la condición sea verdadera.

Ejemplos:


# Bucle for
for i in range(1, 6):
    print(f"Número: {i}")
# Esto imprimirá los números del 1 al 5.

# Bucle while
contador = 1
while contador <= 3:
    print(f"Repetición: {contador}")
    contador += 1
# Esto imprimirá "Repetición: 1", "Repetición: 2" y "Repetición: 3".

Trabajando con Listas y Diccionarios

Python ofrece estructuras de datos flexibles como listas y diccionarios.

  • lista = [1, 2, 3] - Una lista es una colección ordenada de elementos.
  • diccionario = {"clave": "valor"} - Un diccionario es una colección de pares clave-valor.

Ejemplos:


frutas = ["manzana", "plátano", "cereza"]
print(frutas[0])  # Esto imprimirá "manzana"

persona = {
    "nombre": "Juan",
    "edad": 30,
    "ciudad": "Madrid"
}
print(persona["nombre"])  # Esto imprimirá "Juan"
