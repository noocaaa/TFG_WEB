Guía de HTML

¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar para crear páginas web. Es como el esqueleto de una página, determinando su estructura y contenido.

Etiquetas y Elementos

HTML se escribe con etiquetas, que son como pequeñas instrucciones para el navegador. Estas etiquetas le dicen cómo mostrar el contenido.

Ejemplo:


<h1>Esto es un título</h1>
<p>Esto es un párrafo.</p>

El código anterior mostrará un título grande seguido de un párrafo.

Atributos

Los atributos proporcionan información adicional sobre un elemento. Por ejemplo, puedes usar el atributo "href" en la etiqueta <a> para crear un enlace.

Ejemplo:


<a href="https://www.ejemplo.com">Visitar ejemplo.com</a>

Esto creará un enlace que lleva a "ejemplo.com" cuando se haga clic en él.

Listas y Tablas

HTML nos permite crear listas y tablas para organizar la información.

Ejemplos:



<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>


<table border="1">
    <tr>
        <td>Fila 1, Celda 1</td>
        <td>Fila 1, Celda 2</td>
    </tr>
    <tr>
        <td>Fila 2, Celda 1</td>
        <td>Fila 2, Celda 2</td>
    </tr>
</table>

El código anterior creará una lista con dos elementos y una tabla con dos filas y dos celdas por fila.

Formularios

Los formularios en HTML nos permiten recolectar información de los visitantes, como nombres, correos electrónicos y más.

Ejemplo:


<form action="/enviar" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    <input type="submit" value="Enviar">
</form>

El código anterior crea un formulario con un campo de texto para ingresar un nombre y un botón para enviar el formulario.
