Guía de CSS

¿Qué es CSS?

CSS (Hojas de Estilo en Cascada) es el lenguaje que usamos para dar estilo a nuestras páginas web. Con CSS, podemos cambiar colores, fuentes, espacios y más, ¡haciendo que nuestras páginas se vean geniales!

Selectores y Propiedades

En CSS, usamos "selectores" para elegir qué parte de nuestra página queremos estilizar. Luego, usamos "propiedades" para decidir cómo queremos que se vea.

Ejemplo:


p {
    color: rojo;
    font-size: 16px;
}

El código anterior cambiará el color del texto de todos los párrafos a rojo y establecerá su tamaño de fuente a 16 píxeles.

Colores

Hay muchas formas de definir colores en CSS. Puedes usar nombres de colores, códigos hexadecimales, RGB y más.

Ejemplos:


/* Usando nombre de color */
h1 {
    color: blue;
}

/* Usando código hexadecimal */
h2 {
    color: #FF5733;
}

/* Usando RGB */
h3 {
    color: rgb(255, 0, 0);
}

Los códigos anteriores dan color azul al texto de los encabezados h1, naranja a los h2 y rojo a los h3.

Fuentes y Texto

Con CSS, podemos cambiar la fuente, el tamaño, el espaciado y otras propiedades del texto.

Ejemplos:


/* Cambiar la fuente y tamaño */
p {
    font-family: 'Arial', sans-serif;
    font-size: 14px;
}

/* Estilo y decoración del texto */
a {
    text-decoration: none;
    font-weight: bold;
}

El primer ejemplo hace que el texto de los párrafos use la fuente Arial (o sans-serif si Arial no está disponible) y tenga un tamaño de 14 píxeles. El segundo ejemplo hace que los enlaces no tengan subrayado y sean negritas.

Posicionamiento y Espaciado

CSS nos permite controlar el posicionamiento de los elementos en la página y el espacio entre ellos.

Ejemplos:


/* Margen y padding */
div {
    margin: 10px;
    padding: 20px;
}

/* Posicionamiento absoluto */
.caja {
    position: absolute;
    top: 50px;
    left: 100px;
}

El primer ejemplo añade un margen de 10 píxeles alrededor de los div y un relleno de 20 píxeles dentro de ellos. El segundo ejemplo posiciona el elemento con la clase "caja" 50 píxeles desde la parte superior y 100 píxeles desde la izquierda de su contenedor.
