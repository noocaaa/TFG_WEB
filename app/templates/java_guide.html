Guía de Java

Introducción a Java

Java es un lenguaje de programación orientado a objetos ampliamente utilizado, conocido por su portabilidad y robustez. Se usa desde aplicaciones móviles hasta sistemas empresariales.

Operaciones Básicas

Java, al igual que C++ y Python, nos permite realizar operaciones matemáticas básicas.

  • int suma = a + b; - Suma los valores de "a" y "b".
  • int resta = a - b; - Resta "b" de "a".

Ejemplo:


int a = 5;
int b = 3;
int resultado = a + b;  // El resultado es 8

Tomando Decisiones con Condicionales

Java ofrece estructuras condicionales para tomar decisiones en el código.

  • if (condición) { instrucciones } - Si la condición es verdadera, realiza las instrucciones dentro de las llaves.
  • else { instrucciones } - Si la condición del "if" anterior no es verdadera, realiza estas instrucciones.

Ejemplo:


int edad = 12;
if (edad >= 18) {
    System.out.println("Eres un adulto.");
} else {
    System.out.println("Aún eres joven.");
}
// Esto imprimirá "Aún eres joven."

Repetir Instrucciones con Bucles

Java tiene estructuras de bucle que nos permiten repetir instrucciones.

  • for (inicio; condición; actualización) { instrucciones } - Repite las instrucciones mientras la condición sea verdadera.
  • while (condición) { instrucciones } - Continúa repitiendo las instrucciones mientras la condición sea verdadera.

Ejemplos:


// Bucle for
for (int i = 1; i <= 5; i++) {
    System.out.println("Número: " + i);
}
// Esto imprimirá los números del 1 al 5.

// Bucle while
int contador = 1;
while (contador <= 3) {
    System.out.println("Repetición: " + contador);
    contador++;
}
// Esto imprimirá "Repetición: 1", "Repetición: 2" y "Repetición: 3".

Trabajando con Arreglos y Colecciones

Java ofrece estructuras de datos como arreglos y colecciones para trabajar con grupos de objetos.

  • int[] numeros = new int[3]; - Declara un arreglo de enteros.
  • ArrayList lista = new ArrayList(); - Declara una lista de cadenas.

Ejemplos:


int[] numeros = {1, 2, 3};
System.out.println(numeros[0]);  // Esto imprimirá "1"

ArrayList nombres = new ArrayList();
nombres.add("Juan");
nombres.add("Ana");
System.out.println(nombres.get(0));  // Esto imprimirá "Juan"
